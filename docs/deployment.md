# 部署说明

本项目当前处于开源骨架阶段，后续会提供完整的一键部署流程。

## 计划支持

- Docker Compose
- Vercel / Netlify 前端部署
- Node.js API 服务部署
- PostgreSQL
- Redis
- 对象存储

## 环境变量规划

```env
APP_URL=https://moyuanai.com
DATABASE_URL=
REDIS_URL=
OPENAI_API_KEY=
ALIPAY_APP_ID=
ALIPAY_PRIVATE_KEY=
ALIPAY_PUBLIC_KEY=
ALIPAY_NOTIFY_URL=
```
