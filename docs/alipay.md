# 支付宝接入规划

墨元 AI 计划支持支付宝支付，优先接入：

- 电脑网站支付
- 手机网站支付
- 交易查询
- 交易退款
- 异步通知验签
- 订单关闭

## 推荐支付流程

1. 用户选择积分套餐。
2. 服务端创建本地订单。
3. 服务端调用支付宝接口生成支付页面。
4. 用户完成付款。
5. 支付宝向服务端发送异步通知。
6. 服务端验签并确认金额、订单号、支付状态。
7. 给用户账户增加积分。
8. 记录充值流水和支付流水。

## 注意事项

- 不要只依赖前端跳转结果确认支付。
- 异步通知必须验签。
- 订单金额、商户订单号和支付状态必须二次校验。
- 积分充值逻辑需要幂等，避免重复回调导致重复充值。
